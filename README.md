# Doctor Lookup
## by Amy Hwang

# description of the webpage
This webpage is for anyone to search doctors by their medical issue(s). When searched, doctor information such as name, specialties, location, and contact info is shown.

#Technologies Used
* NodeJS
* Gulp
* Bower
* jQuery
* Sass
* BetterDoctor API
